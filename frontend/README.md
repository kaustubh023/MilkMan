# KP Fresh Dairy Frontend

This folder contains the React + Vite frontend for the KP Fresh Dairy project.

For complete project documentation, backend setup, routes, and login details, see:

- `../README.md`

## Quick Start

```powershell
cd d:\Dudhvala\frontend
npm install
npm.cmd run dev
```

Frontend runs on:

```text
http://127.0.0.1:5173
```

## Build

```powershell
npm.cmd run build
```

## Lint

```powershell
npm.cmd run lint
```
